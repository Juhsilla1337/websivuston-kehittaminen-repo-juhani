Siirrymme responsiivisen kuvan kanssa spedeilemiseen
Voimme huomata, että tälläinen kuva kannattaa pyrkiä luomaan mahdollisimman isolla resoluutiolla.
Tai SVG eli vektorikuvana jolloin kuvan laatu ei sakene kuvan suurentuessa normaalikokoaan isommaksi.
Kuten CSS tiedostossa näkyy, kuva on asetettu täyttämään koko leveytensä,
mutta korkeudesta ei ole määritetty mitään, jolloin se skaalautuu automaattisesti kuvan alkuperäisen mittasuhteen mukaiseksi.

Tämäkin on ihan siisti CSS ominaisuus.
